Mobile Phone Replica

A university assignment to create a Figma replica of a mobile application from a website that doesn't have a native mobile app. This project was designed to train proficiency in Figma while cultivating a deep understanding of User Experience design principles and presentation skills.

Figma UX Design UI Design Prototyping SUS Method User Research
Key Highlights
  • Converted a desktop-only website into a full mobile application prototype in Figma
  • Analyzed and restructured information architecture for mobile-first user flows
  • Designed interactive prototypes with realistic navigation and transitions
  • Conducted usability testing using the System Usability Scale (SUS) methodology
  • Presented the final design and findings to peers, developing presentation skills
  • Gained hands-on experience bridging UX research with visual design decisions
