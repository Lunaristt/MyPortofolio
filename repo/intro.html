Introduction to Figma

A university project assigned by my lecturer to create an introduction to Figma as a beginner. The task was to duplicate my personal mobile phone's home screen and UI elements as a reference exercise, building foundational skills in UI design and prototyping tools.

Figma UI Design Prototyping Mobile Interface Software Development
Key Highlights
  • First introduction to Figma as a design and prototyping tool
  • Replicated a mobile phone home screen with pixel-level accuracy
  • Learned core Figma concepts: frames, components, auto-layout, and constraints
  • Developed attention to detail in spacing, typography, and color matching
  • Built a foundation for future UI/UX design coursework and projects
